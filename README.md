# 📱 MGRS Calc (MGRS 座標計算・戦術マップPWA)

本アプリは、電波のないオフライン環境（演習場や山間部など）でも動作する、実戦的なMGRS座標計算およびマップ可視化ツールです。
PWA（Progressive Web Apps）として開発されており、スマートフォンにインストールしてネイティブアプリのようにサクサクと動作します。

## ✨ 主な特長

* **完全オフライン動作**: キャッシュを利用し、圏外でも全機能が使用可能。
* **🌙 タクティカル・ナイトモード**: 暗順応を妨げない「赤・黒」の専用UIをワンタップで起動。
* **📍 GPS連動**: スマホのGPSを利用し、現在地のMGRS座標（E/N 下5桁）と標高を自動取得。
* **一括データ共有機能**: 登録した座標データをテキスト形式で一括入出力でき、LINEやメモ帳経由での部隊内共有が容易。

## 📲 インストール手順（使い方）

アプリストアを経由せず、ブラウザから直接ホーム画面に追加して使用します。

### iPhone (Safari) の場合
1. Safariで本アプリのURLを開く
2. 画面下部の「共有ボタン（四角から矢印が上に出ているアイコン）」をタップ
3. **「ホーム画面に追加」**を選択
4. ホーム画面に追加されたアイコンから起動

### Android (Chrome) の場合
1. Chromeで本アプリのURLを開く
2. 画面右上の「︙（メニュー）」をタップ
3. **「ホーム画面に追加」**を選択

## 🛠 各機能の概要

### 1. 座標対座標
現在地と目標地の座標から、**方位角（mil）・距離（m）・高低差（m）・高低角（mil）**を計算します。標高(Z)は未入力でも平面計算が可能です。

### 2. 極座標
現在地から目標への諸元（方位角・距離・高低角）を入力し、**目標の座標と標高**を算出します。算出結果はワンタップで名前や色をつけて登録できます。

### 3. 前方交会法
離れた2つの観測所（既知点）から、同一目標を観測した際の方位角を入力し、**目標の座標（交点）**を算出します。

### 4. 後方交会法
自分の位置がわからない時、マップ等で座標が判明している2つの目標（山頂や鉄塔など）への方位角を測定・入力することで、**現在の自分の座標**を逆算します。

### 5. トラバース計算
開始点から「方位〇〇へ距離〇〇m移動」という連続した移動ルートを複数入力し、**すべての経由点の座標を一括で算出・登録**します。

### 6. 座標の登録・一覧・GPS
よく使う陣地、観測所、目標の座標を管理・編集する機能です。

**【テキスト一括入出力（部隊間データ共有）】**
以下の書式（スペース区切り）でテキストエリアに貼り付け、「↓ 読み込み」を押すと一括でマップに登録されます。同名データは自動で最新に上書きされます（陣地変換時などに便利です）。

* **書式:** `名称 E座標 N座標 標高(任意) 色`
* **入力例（標高あり）:** `FO1 12345 67890 150 緑`
* **入力例（標高なし）:** `CB401 12350 67900 赤`

### 7. マップ (1km方眼)
登録した座標を、1km方眼の戦術マップ上にピンで可視化します。
* 1本指でスライド移動、2本指でピンチ（拡大・縮小）操作。
* 画面下のプルダウンから「開始点」と「終了点」を選ぶと、その2点間の方位角・距離・高低差・高低角を即座に計算し、マップ上に照準ラインを描画します。

## 🔄 アプリのアップデートについて
開発者がプログラムを更新した場合、アプリを完全に終了（タスクキル）して数分待ってから再度開くと、Service Workerの働きにより自動的に最新版にアップデートされます。キャッシュを手動で削除する必要はありません。
